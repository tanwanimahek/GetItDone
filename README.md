# ✅ GetItDone

A minimal and aesthetic To-Do app made using HTML, CSS, and JavaScript - designed to help you stay productive with a clean, calming UI and smooth task management.

> Made with ❤️ by [Mahek Tanwani](https://github.com/tanwanimahek)

---

## 🔥 Features

- **Add tasks** via input field or pressing `Enter`
- **Mark as complete** with a satisfying checkbox toggle
- **Delete tasks** with a single click
- **Inline editing** — double-click any task to rename it
- **Filter tasks** — view All, Active, or Done tasks
- **Live stats** — see Total, Completed, and Remaining counts update in real time
- **localStorage** — your tasks are saved and survive page refreshes
- **Light / Dark mode** toggle 🌙☀️
- **Empty state** message when no tasks exist
- **Shake feedback** when trying to add an empty task
- **Fully responsive** — desktop sidebar layout + mobile-friendly stacked view
- **Smooth animations** — soft transitions and slide-in effects throughout

---

## 🛠️ Tech Stack

- HTML5
- CSS3 (CSS Variables, Flexbox, Keyframe Animations, Media Queries)
- JavaScript (DOM Manipulation, localStorage, Events)

## 💡 Concepts Covered

This project was built to demonstrate a solid foundation of front-end fundamentals:

| Concept | Where it's used |
|---|---|
| DOM Manipulation | Creating, updating, and deleting task elements |
| Event Listeners | click, keypress, dblclick, blur |
| localStorage | Saving and loading tasks across sessions |
| CSS Variables | Theming and dark mode |
| Responsive Design | Media queries for desktop and mobile |
| Input Validation | Shake animation on empty submit |
| Conditional Rendering | Filter logic and empty state |
| Array Methods | `.filter()`, `.indexOf()`, `.splice()` |

## 🚀 How to Run

1. Clone or download the repository
2. Open `todo-dom.html` in any browser
3. No dependencies, no build step — it just works

```bash
git clone https://github.com/tanwanimahek/GetItDone.git
cd GetItDone
# open todo-dom.html in your browser
```
